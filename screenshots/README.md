## How to Take Screenshots

### Using Android Studio Emulator:
1. Run your app in the emulator
2. Click the camera icon in the emulator toolbar
3. Save the screenshot
4. Copy it to this `screenshots/` directory

### Using Physical Device:
1. Connect your device via USB
2. Take screenshots using your device's screenshot function
3. Transfer to your computer
4. Copy to this `screenshots/` directory
